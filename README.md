## Overview

SysDoc is an open-source system diagnostics toolkit written in Python.

It provides comprehensive information about hardware, operating system, network, storage, battery, temperatures, internet connectivity and much more through a modern command-line interface.

Designed for:

- 🐧 Linux
- 🪟 Windows
- 🍎 macOS

---

## Features

- CPU Monitoring
- Memory Analysis
- Disk Health
- SMART Status
- Battery Information
- Hardware Temperatures
- Internet Speed Test
- Ping Test
- DNS Diagnostics
- Network Interfaces
- Open Ports Scanner
- Running Processes
- System Information
- Export Reports
- JSON Reports
- HTML Reports
- TXT Reports
- Beautiful Terminal UI
- Cross Platform

## Installation

```bash
git clone https://github.com/Santosxbk/SysDoc.git

cd sysdoc

pip install -r requirements.txt
```
